# Code of Conduct

As contributors and maintainers of this project, we pledge to respect everyone who contributes by:

- Using welcoming and inclusive language  
- Being respectful of differing viewpoints and experiences  
- Gracefully accepting constructive criticism  
- Focusing on what is best for the community  
- Showing empathy towards other members

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by contacting the repository owner.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org).

For assignment-specific guidelines, please also review the [README.md](./README.md) which outlines expectations for each homework branch.
